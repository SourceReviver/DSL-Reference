Справочник по DSL

Lingvo x5 (15.0)

Хроника

Год выпуска: 2011.

Данная версия является «рекордсменом» по недокументированным нововведениям и изменениям.

Без упоминания в документации введены в употребление новые тэги [br] и [trn1], атрибут target в тэгах [ref] и [url], директива предварительной обработки #ICON_FILE.

Так же недокументированно появились тэги [video] и [preview], однако на этом их история пока закончилась, развития они не получили и по-видимому были отвергнуты в процессе разработки.

Возможности DSL Compiler пополнились режимом пакетной обработки DSL из командной строки. Известно это стало из сообщения специалиста службы тех-поддержки ABBYY на форуме компании в ответ на просьбы пользователей. При этом не все параметры были упомянуты и о существовании части из них пользователи узнавали самостоятельно.

Без упоминания в документации добавлена поддержка языков: азербайджанский, киргизский, таджикский, туркменский, узбекский. Эти языки поддерживаются по обоим атрибутам name и id. Поддержка башкирского языка добавлена только по атрибуту name (атрибут id для башкирского вызывает протест компилятора).

Недоразумения в поддержке латинского языка, появившиеся в x3 (см. подробнее в Истории x3), в x5 имеют место до версии 15.0.511.0. Начиная с 15.0.826.5 в атрибуте id начинает поддерживаться идентификатор 1142, а идентификатор 1540 объявляется ошибочным. Несмотря на это в системных латинских словарях продолжает использоваться идентификатор 1540. Исправление этой ситуации происходит в версии 15.0.826.26.

Прекращена поддержка Немецкого языка новой орфографии (GermanNewSpelling, 32775), который поддерживался в Lingvo начиная с версии 8.0.
