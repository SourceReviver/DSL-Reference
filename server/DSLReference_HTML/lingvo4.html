Справочник по DSL

Lingvo 4.0

Хроника

Год выпуска: 1993. Может работать в двух режимах: DOS и Windows.

Официально программа работает со словарями двух языковых направлений: англо-русского и русско-английского.

Компилятор пользовательских словарей LC.COM работает только из командной строки. Также имеется демонстрационный словарик, где в служебных комментариях указываются следующие возможные конструкции DSL:

  • заголовки словом и фразой,
  • несколько заголовков у одной статьи,
  • несортируемая часть заголовка,
  • альтернативная часть заголовка,
  • создание дочерних статей (вложенных подкарточек),
  • команда клонирования заголовка,
  • тэг служебных комментариев.

В краткой справке по использованию LC.COM (отображаемой в окне консоли) мы видим команды для компилирования англо-русского словаря с расширением *.adc и русско-английского словаря с расширением *.bdc. При подключении скомпилированных тестовых словарей видим имена их файлов с прибавлением суффикса -E для англо-русских и -R для русско-английских. Отсюда очевидно, что языковое направление словаря определялось по его расширению. Вполне вероятно, что в основу расширений положены аббревиатуры A Dictionary Cards (карточки словаря категории A, т.е. англо-русского) и B Dictionary Cards (карточки словаря категории B, т.е. русско-английского).
